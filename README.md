# Server-sent events using FastAPI

![Progress Steps Example](/example.gif)

## Pre-requisites

- Python 3.10
- Poetry

## Installation

```bash
poetry install
```

## Usage

```bash
poetry run uvicorn main:app --reload
```

You can then access this excample at [http://127.0.0.1:8000](http://127.0.0.1:8000)
